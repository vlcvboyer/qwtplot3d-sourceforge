Version 0.2.0,  24 September 2003



           (CVS: Changes in CVS only)

Features

  • Scaling, rotating, shifting, zooming of data sets and mathematical functions ( see Data Input Topics )
  • Interactive mouse handling
  • Resolution changing on the fly
  • Reading from MESH data files :See the 'examples/data' directory and NativeReader documentation for this . You will find example datasets here
  • Data structures providing nodes and triangulation
  • Reading from FEM files
  • Flat (2D) labeling
  • Wireframe, filled polygons, hidden line
  • Floor projections (Isolines, Data)
  • Autoscalable axes
  • Free color model
  • Flat/Gouraud shading 
  • Color legends
  • Pixmap output (all Qt supported formats)
  • Vector output (PostScript,  EPS and PDF)  via gl2ps      // This feature  is more thoroughly discussed  here
